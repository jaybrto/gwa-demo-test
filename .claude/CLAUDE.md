# GWA Demo Repository

This is a test repository for GitHub Workflow Agents (GWA) end-to-end testing.

## Project Context
- **Purpose:** E2E testing of GWA automation
- **Tech Stack:** TypeScript, Bun
- **Infrastructure:** K3s with Longhorn storage

## Testing Goals
1. Verify GitHub Projects integration
2. Test column-based workflow triggers
3. Validate multi-agent swarm architecture
4. Ensure planning templates work correctly

## Git Workflow
- Feature branches: `claude/issue-{N}-{description}`
- Conventional commits required

## When Working on Issues
1. Read the issue description carefully
2. Create implementation plan
3. Implement the solution
4. Write tests
5. Create PR with clear description
